Solution: Open data standards.

Three main types of open data standards:
  • Schematic: Defines structure of the data to be published. This includes the names, descriptions, and data types of data fields or columns.
  • Semantic: Defines the terminology or language in the data which is published. E.g. define the terms "Arson" and "Robbery"
  • Atomic: Defines how basic elements of data must be represented. E.g., a date and time should be formatted as “2017-01-01T13:00:00Z” OPEN DATA DIRECTORY (datastandards.directory/glossary)

We will focus on IFC.
IFC is a standardized, digital description of the built environment. It is an open, international standard (ISO 16739-1:2018), meant to be vendor-neutral, or agnostic, and usable across a wide range of hardware devices, software platforms, and interfaces for many different use cases.
◼︎ Definition of IFC taken from BuildingSMART's website. ◼︎ Important to note that IFC is an open standard.

Native OpenBIM



◼︎ Native OpenBIM or NativeIFC refers to programs that have IFC as their fundamental data model. ◼︎ No exporting, importing, or translating of data. We model directly in IFC. ◼︎ We will focus on BlenderBIM.

Blender BIM

  • Aug 2019: 83 lines of code written to export Blender meshes to IFC
  • Oct 2019: Packaged and released to the public
  • Jan 2020: Community user automates visualization of compactor machines
  • Nov 2020: Swiss army knife of free BIM utilities in production use on commercial projects
◼︎ 2019, 83 lines of code to export Blender meshes to IFC. ◼︎ Thomas had written an importer. Compactor machines. ◼︎ Actively developed since 2020, over 100 contributors to the code. ◼︎ Complete re-write so all geometry is native IFC geometry to the extent possible (parametric) and not Blender meshes.

Blender BIM

◼︎ All IFC functionality comes from IfcOpenShell. ◼︎ IfcOpenShell closely follows IFC, doesn't deviate much from it, but adds many more features. ◼︎ BlenderBIM is largely a UI for IfcOpenShell. ◼︎ BlenderBIM's geometry is separated from Blender -> two files (IFC and Blender).
